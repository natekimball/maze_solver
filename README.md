### Find all solutions to a maze
```
$ cargo run . -f resources/13x7maze.txt
```
### Find the shortest path in a maze
```
$ cargo run . -f resources/13x7maze.txt -s
```
### Find any solution to a maze
```
$ cargo run . -f resources/13x7maze.txt -a
```
